
## atom 
没有括号括起来就是


## list 
() 括起来的就是

## S-expression
atom 和 list 都属于 S-expression

## car

(car X) = P

X is must be no-empty list
p is any S-expression,

## cdr
(cdr X) = P

X is must be no-empty list
P is a list

## cons

(cons X Y) = P

X is any S-expression,
Y is must be a list
p is a list.

	// 其实就是把 X 插入 到 Y 最前面

## null?
(null? X) = Y[true/false]
X is must be a list
if x non-empty list, Y is false. 
if x empty list, Y is true.
